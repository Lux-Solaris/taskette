* 项目说明

- 用 ~Ruby~ 的 ~Sinatra~ 框架, 开发一个任务跟踪看板

** 设计功能

- 可以跟踪一些带标签的任务, 对任务进行附加信息 *Mark*, 以及附带的番茄钟.

- 一个任务跟踪系统, 包括以下功能
  + 任务看板, 一目了然;
  + 任务增加, 修改, 删除; 任务是否完成跟踪;
  + 任务 DDL 跟踪, 超期限提醒.

- 可以给任务附加额外信息, 比如
  + 标签, 并且可以根据标签进行分类;
  + 附加信息, 可以给任务添加一些额外的提示或记录;
  + 任务完成时间, 可以进行一些基础的统计.

- 任务聚焦界面
  + 进行任务的聚焦, 可以使用

** 开发阶段

+ 基本完成所有功能.

* 部署说明

- 请确保你有相对好的网络连接; 好的网络连接 *不包括* 科学上网

  + /使用 *[[https://gems.ruby-china.com/][Ruby-China]]* 镜像源, 因此无需科学上网/

- 请确保本地拥有 Ruby: ~bundle~, 若没有, 请自行安装.

- 安装倚赖包
  #+begin_src bash
    bundle install
  #+end_src

- 启动服务
  + 直接使用 ~Ruby~ 进行启动
    #+begin_src bash
      bundle exec ruby app.rb
    #+end_src
  + 或者使用 ~Rackup~ 进行启动
    #+begin_src bash
      bundle exec rackup
    #+end_src
